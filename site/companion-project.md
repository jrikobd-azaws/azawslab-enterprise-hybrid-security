# Companion Project: local-ai-lab-infra

The companion project extends the portfolio into local-first AI infrastructure and agentic workflow experimentation.

## Ecosystem relationship

| Repository | Role |
|---|---|
| `azawslab-enterprise-hybrid-security` | Enterprise platform, security architecture, IaC, evidence vault, O6 enclave pattern |
| `local-ai-lab-infra` | Local-first AI lab, RAG, multi-agent pipeline, provider routing, local model execution |

## Why it matters

Together, the repositories show both sides of modern platform work:

- Secure enterprise foundation.
- Governed operational controls.
- AI-assisted workflows.
- Local-first experimentation.
- Evidence-backed engineering discipline.
