# Release 3 - Multi-Cloud Kubernetes, GitOps and DevSecOps Roadmap

Release 3 is the roadmap for extending the platform into multi-cloud Kubernetes delivery, GitOps operations and DevSecOps governance. It is intentionally labelled as roadmap scope unless a capability is already implemented or evidenced in earlier releases.

## What this roadmap targets

| Capability | Roadmap signal |
|---|---|
| AKS and EKS direction | Multi-cloud Kubernetes operating model |
| GitOps delivery | Argo CD, reconciliation and controlled deployment patterns |
| DevSecOps controls | Image assurance, policy-as-code and admission control |
| Observability and resilience | Monitoring, logging, tracing, recovery and chaos validation direction |
