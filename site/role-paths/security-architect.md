# Security Architect Path

## Security architecture themes

- Identity-first controls.
- Separation of duties through Terraform roots and workflows.
- Private platform patterns for AKS and AVD.
- Hybrid and multi-cloud routing with explicit inspection paths.
- Evidence redaction and public-safe proof handling.
- Governed AI operations with policy mediation and human approval.

## Best review path

1. Architecture document.
2. Evidence guide.
3. Private AKS and AVD section.
4. O6 AI operations boundary.
5. Terraform state and pipeline map.
