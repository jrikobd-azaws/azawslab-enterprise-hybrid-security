A2 AWX VM Module Reuse Audit
=============================

Purpose:
Review whether vm-dev-awx-01 can reuse the existing VM/compute module and platform-management pattern before Terraform code is changed.

Required AWX alignment:
- Ubuntu OS, same as management-host pattern where possible
- Same approved VM SKU policy, expected Standard_B2als_v2 unless current management host proves otherwise
- Same mandatory tag model
- Same management network/subnet pattern as vm-dev-mgmt-01
- No accidental public exposure
- Root/state: terraform/platform-management/dev / platform-management-dev.tfstate
