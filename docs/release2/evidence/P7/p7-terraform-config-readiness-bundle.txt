## Evidence safety

The detailed generated bundle is intentionally retired from public review because it was too noisy for portfolio navigation and contained stale generated citation markers.

No Terraform state files, binary plan files, access tokens, private keys, credentials, kubeconfigs, or sensitive tenant details should be published in this evidence folder.
