P3 Terraform correction plan - resource group mandatory tag enforcement

Finding:
- Existing mandatory tag assignments used the built-in policy Require a tag on resources.
- That policy is Mode: Indexed and did not deny untagged resource group creation.
- Azure Policy discovery confirmed the correct built-in for resource groups is Require a tag on resource groups, Mode: All.

Terraform correction:
- Added resource-group-level mandatory tag assignments in terraform/governance/main.tf:
  - pa-rgtag-env
  - pa-rgtag-proj
  - pa-rgtag-own
  - pa-rgtag-cost

Local validation:
- terraform fmt -check passed.
- terraform init succeeded.
- terraform validate succeeded.
- terraform plan showed 4 to add, 0 to change, 0 to destroy.

Deployment path:
- Do not apply locally.
- Submit via PR and deploy through GitHub Actions controlled Terraform Apply.
