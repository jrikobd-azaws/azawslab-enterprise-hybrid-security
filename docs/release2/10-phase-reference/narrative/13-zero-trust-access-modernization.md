## Future Access Modernization - GSA / ZTNA

Entra Global Secure Access / ZTNA is deferred as a future access-modernization enhancement.

It is no longer the active O4 scope. The active O4 scope is Private AKS and the active O5 scope is AVD secure admin/dev workspace.

Future GSA/ZTNA work can be revisited after:
- O3b AWS Cisco routing is validated
- O3c transitive routing is validated
- O4 private AKS is validated
- O5 AVD secure workspace is validated
